
## Actividad 1: Secuancia de led

Este es un código básico para una secuencia de 8 LEDs. Puedes modificarlo y adaptarlo a tus necesidades.

```cpp
const int ledPins[8] = {2, 3, 4, 5, 6, 7, 8, 9}; // Pines de los LEDs

void setup() {
  for (int i = 0; i < 8; i++) {
    pinMode(ledPins[i], OUTPUT);
  }
}

void loop() {
  for (int i = 0; i < 8; i++) {
    digitalWrite(ledPins[i], HIGH);
    delay(100); // Duración de cada LED encendido
    digitalWrite(ledPins[i], LOW);
  } 

  delay(500); // Retardo entre secuencias
}
```

### Explicación:

- ledPins: Arreglo que define los pines digitales donde se conectan los LEDs.
- setup: Configura los pines de los LEDs como salida.
- loop: Recorre el arreglo ledPins encendiendo y apagando cada LED uno por uno.
- digitalWrite(ledPins[i], HIGH): Enciende el LED en la posición i.
- delay(100): Espera 100 milisegundos (tiempo que el LED permanece encendido).
- digitalWrite(ledPins[i], LOW): Apaga el LED en la posición i.

### Secuencia alternativa:

- Puedes modificar el tiempo de encendido de cada LED ajustando el valor de delay(100).
- Puedes modificar el tiempo de espera entre secuencias ajustando el valor de delay(500).
- Puedes agregar más secuencias de LEDs al código.
- Puedes usar diferentes patrones para encender y apagar los LEDs.
